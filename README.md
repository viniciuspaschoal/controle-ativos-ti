# Controle de Ativos TI

Projeto para controle, conferência e auditoria de ativos de TI.

## Estrutura

- `python/`: protótipos, importador Excel -> SQLite e lógica validada
- `kotlin/`: nova implementação em Kotlin, visando futura base do app
- `docs/`: documentação do projeto

## Funcionalidades já prototipadas

- Importação de planilha Excel para SQLite
- Consulta de ativos e monitores
- Registro de conferências
- Resumo consolidado de leituras
- Auditoria de ativos com validação de monitores
- Visualização das auditorias
